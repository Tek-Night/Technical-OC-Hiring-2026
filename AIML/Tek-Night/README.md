# Dataset Setup

This project uses the **DIV2K** high-resolution (HR) dataset.

## Download

1. Visit the official DIV2K website:

   https://data.vision.ee.ethz.ch/cvl/DIV2K/

2. Download:

   - `DIV2K_train_HR.zip`

3. Extract the downloaded archive.

## Folder Structure

After extracting, copy the images into the following directory:

```text
project-files/
└── dataset/
    └── raw_hr/
        ├── 0001.png
        ├── 0002.png
        ├── 0003.png
        └── ...
```

The `raw_hr` folder should contain **800 PNG images**.

## Create Train / Validation / Test Split

Run:

```bash
python data/split_dataset.py
```

This automatically creates:

```text
dataset/
├── train/
├── val/
└── test/
```

using a **700 / 50 / 50** split.

> **Note:** Only the high-resolution images are used. Do **not** download or use the pre-generated LR (low-resolution) versions of the DIV2K dataset. Low-resolution images are generated by this project during training using the implemented degradation pipeline.
