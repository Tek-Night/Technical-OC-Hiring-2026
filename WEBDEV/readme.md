# Web Development Head

## Task

Build an Enigma Event Website with Admin Dashboard.

Build a functional website for an Enigma event where participants can view event details, register, and where organizers can manage registrations from a simple admin dashboard.

This is a very strong task because it tests frontend, backend, forms, routing, UI design, data handling, deployment, and real club usefulness.

### Success Criteria
- Build a working event website with clear event information and registration flow.
- Implement a simple admin dashboard for viewing and managing registrations.
- Show both frontend and backend understanding where relevant.
- Use clean UI design, navigation, and basic data handling.
- Explain the architecture and workflow clearly enough for someone else to recreate it.

### Technical Demo Video
A short technical demo video is necessary for this task. The video should demonstrate the website in action and explain the main features and implementation choices. During the interview, we will ask about the project, your design decisions, and how you would explain it to someone who wants to rebuild it from scratch.
